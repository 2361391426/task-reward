## ⚠️ 注意事项

### 1. 本地 API 地址
小程序使用的是 `http://localhost:3001`，只能在开发工具中测试。
真机测试需要：
- 部署后端到服务器
- 修改 API 地址为服务器地址
- 在微信小程序后台配置服务器域名

### 2. 图片上传
- 开发工具中可以选择本地图片
- 真机测试可以拍照或从相册选择

### 3. 微信登录
- 开发工具中会模拟登录
- 真机测试需要真实的微信账号

### 4. 域名校验
- 开发时必须关闭域名校验
- 发布时需要配置合法域名

## 🐛 常见问题

### Q: 编译失败
A: 检查 node_modules 是否完整
```bash
cd F:\task-reward-miniapp
npm install
npm run dev:mp-weixin
```

### Q: 页面空白
A:
1. 检查控制台是否有错误
2. 检查 API 地址是否正确
3. 检查后端服务器是否运行

### Q: 无法连接后端
A:
1. 确认后端服务器运行在 http://localhost:3001
2. 确认关闭了域名校验
3. 查看 Network 标签的请求

### Q: 图片上传失败
A:
1. 检查后端 uploads 目录是否存在
2. 检查上传接口是否正常
3. 查看控制台错误信息
