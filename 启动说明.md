## 如何在微信开发者工具中运行

1. 打开微信开发者工具
2. 点击"导入项目"
3. 选择目录: `F:\task-reward-miniapp\dist\dev\mp-weixin`
4. AppID: 使用测试号或选择"测试号"
5. 点击"导入"
6. 在"详情" -> "本地设置"中，勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"

**注意**: tabBar 图标已创建在 `static/tabbar/` 目录，如果遇到图标未找到的错误，请重新编译：
```bash
npm run build:mp-weixin
```

## 常见问题

### 1. 小程序无法连接后端
- 检查后端服务是否运行: `curl http://localhost:3001/api/health`
- 在微信开发者工具中，点击"详情" -> "本地设置" -> 勾选"不校验合法域名"

### 2. 图片上传失败
- 确保 task-reward-backend/uploads/ 目录存在
- 检查后端日志: `cat task-reward-backend/server.log`

### 3. 商家端无法登录
- 检查商家端服务是否运行
- 默认账号: admin / admin123

## 停止服务

```bash
# 停止所有 node 进程
ps aux | grep node | grep -v grep | awk '{print $2}' | xargs kill

# 或者分别停止
kill <PID>  # 使用 ps aux | grep node 查看 PID
```

## 重新启动

```bash
# 启动后端
cd task-reward-backend
node server-local.js > server.log 2>&1 &

# 启动商家端
cd merchant-admin
npm run dev > merchant.log 2>&1 &

# 启动小程序编译
cd ..
npm run dev:mp-weixin > miniapp-dev.log 2>&1 &
```

## 技术栈

- **小程序**: uni-app + Vue 3
- **后端**: Node.js + Express + JSON 文件存储
- **商家端**: Vue 3 + Vite + Element Plus
- **认证**: JWT Token
- **加密**: AES-256-CBC (手机号)、bcrypt (密码)
